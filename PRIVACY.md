# Privacy Policy

This plugin does not collect, store, or transmit any user data.

All operations run locally on the user's machine. The plugin instructs Claude Code to fetch Wikipedia articles using curl, which sends HTTP requests directly from the user's machine to Wikipedia's servers. No intermediate servers or third-party services are involved.

The only network requests made are to `en.wikipedia.org`, subject to [Wikipedia's privacy policy](https://foundation.wikimedia.org/wiki/Policy:Privacy_policy).
